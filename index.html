因数分解 #2

因数・共通因数とは

はじめに「因数」と「共通因数」２つの用語を覚えておきましょう

掛け算で繋がっている式にかけられている数一つ一つを、「因数（いんすう）」といいます

例えば、3(a + b)では、3と(a + b)が因数

(x + 1)(x + 3)では、(x + 1)と(x + 3)が因数

ということになります

そして、すべての項に共通している因数を、「共通因数」といいます

例えば、3a + 3bでは、3が3aと3bの両方ともに共通して掛けられていますよね？

だから3が共通因数ということになります

a2+ab+acでは、aがa2とabとacの全てに共通して掛けられていますよね？

だからaが共通因数ということになります

この共通因数の概念がこの先非常に大切なので、共通因数を見つける練習をしましょう

  • 次の式の共通因数を求めよ（例えばa2と書きたいときは、a^2と書いてください）
  • （1）2ax + 4a
  • （2）9a2b3-21b2
  • （3）(x - y)a - (x - y)b
